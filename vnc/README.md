# VNC Containers for NetLogo

## Installation Instructions

### Prerequisites

A recent version of `docker` (>= 1.12.3) and `docker-compose` (>= 1.8.0)

```
./build.sh
```

## Run Instructions

```
docker-compose up netlogo53
```

For NetLogo 5.3

or 

```
docker-compose up netlogo52
```

For NetLogo 5.2
